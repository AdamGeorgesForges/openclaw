# Bootstrap preset files

These files are optional operator-managed presets for first-run workspace seeding.

- `agents.md` → `AGENTS.md`
- `soul.md` → `SOUL.md`
- `identity.md` → `IDENTITY.md`
- `user.md` → `USER.md`
- `heartbeat.md` → `HEARTBEAT.md`
- `bootstrap.md` → `BOOTSTRAP.md` (disabled by default in preconfigured mode)

Use `agents.defaults.bootstrapPreset.baseDir` with relative file paths in config.
